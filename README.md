# Big Bash League Dataset (2011–2025)

A detailed ball-by-ball dataset covering **618 Big Bash League (BBL)** matches played between **2011 and 2025**.
This dataset includes delivery-level events, runs, wickets, extras, player actions, and full match metadata—ideal for cricket analytics, modeling, visualization, and machine learning projects.

## Features
- 618 BBL matches
- Ball-by-ball delivery data
- Runs, wickets, extras, dismissals
- Batting, bowling, and fielding details
- Match metadata (teams, venue, date, toss, result)

## Format
Available in **JSON** and **YAML** formats.
